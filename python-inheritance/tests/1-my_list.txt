SUCCESS CASES:


Try to append integers in list:

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.append(4)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.append(6)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5, 6]



Try to append negative numbers in list:

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.append(-3)
>>> my_list.print_sorted()
[-3, 1, 2]



Try with just one element to print:

>>> my_list = MyList()
>>> my_list.append(700)
>>> my_list.print_sorted()
[700]



Try to print an empty list:

>>> my_list = MyList()

>>> my_list = MyList()

>>> my_list.print_sorted()
[2, 6, 9]
